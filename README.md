# Node.js Code Generator (Paw Extension)

A [Paw Extension](http://luckymarmot.com/paw/extensions/) that generates Node.js 4.x (without dependencies) code.

## Installation

- Checkout this repo
- type 'make install' 


## Development

### Build & Install

```shell
make install
```

### Watch

During development, watch for changes:

```shell
npm install 
npm run watch
```
